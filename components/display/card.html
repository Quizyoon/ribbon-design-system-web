Components > Display > Card

Card

Cards contain content and actions about a single subject.

Variants

Elevated

Use elevated cards for content that needs visual prominence with shadow.

Outlined

Use outlined cards for content that needs clear boundaries without shadow.

Filled

Use filled cards for subtle content grouping with a tinted background.

Usage

When to use

Use cards to group related information and actions in a contained, scannable format.

Padding

Choose padding size based on content density: none for images, small for compact layouts, medium for standard content, large for spacious layouts.

Clickable

Enable clickable state when the entire card should respond to user interaction.
