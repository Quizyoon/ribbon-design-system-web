Button v1.2.0

버튼을 통해 사용자는 한 번의 탭으로 작업을 수행하고 선택할 수 있습니다.

시작하기

1

Button 컴포넌트를 import 합니다.

JS ES Module 
import { CButton } from '@linecorp/cuetify'
VUE Vue SFC 
<template>
  <CButton @click="handleClick">선물하기</CButton>
</template>

<script setup>
import { CButton } from '@linecorp/cuetify'

const handleClick = () => {
  console.log('clicked')
}
</script>
2

가장 기본적인 Button 사용 방법입니다.

VUE Basic Usage 
<CButton>선물하기</CButton>
3

variant prop을 사용하여 다양한 스타일의 버튼을 만들 수 있습니다.

VUE Variants 
<CButton>선물하기</CButton>
<CButton variant="neutralSolid">쿠폰받기</CButton>
<CButton variant="criticalSolid">삭제하기</CButton>
4

다양한 스타일과 기능을 조합하여 사용할 수 있습니다.

VUE Advanced Usage 
<!-- Ghost 버튼 -->
<CButton variant="ghost">취소</CButton>

<!-- Outline 버튼 -->
<CButton variant="outline">더보기</CButton>

<!-- 아이콘(솔리드) + 텍스트 버튼 -->
<CButton icon="gift-solid">선물하기</CButton>

<!-- 텍스트 + 아이콘 버튼 -->
<CButton variant="neutralWeak" icon="arrow-right" iconPosition="right">사용하기</CButton>

<!-- 아이콘만 있는 버튼 -->
<CButton variant="outline" layout="iconOnly" icon="share-ios" />

Properties

Prop Type
variant? "brandSolid" | "brandWeak" | "neutralSolid" | "neutralWeak" | "criticalSolid" | "criticalWeak" | "outline" | "ghost"
Default "brandSolid"

버튼의 시각적 스타일을 설정합니다.
size? "xsmall" | "small" | "medium" | "large" | "xlarge"
Default "medium"

버튼의 크기를 설정합니다.
layout? "withText" | "iconOnly"
Default "withText"

버튼의 레이아웃 타입을 설정합니다.
icon? string
Default null

버튼 왼쪽에 표시할 아이콘 이름입니다.
iconPosition? "left" | "right"
Default "left"

아이콘의 위치를 설정합니다.
iconFill? boolean
Default true

아이콘의 채우기(fill) 스타일을 설정합니다.
loading? boolean
Default false

로딩 상태를 표시합니다. 활성화 시 버튼이 비활성화됩니다.
disabled? boolean
Default false

버튼을 비활성화합니다.
flexGrow? true | 0 | 1 | number
Default undefined

flex-grow 값을 설정합니다. true면 1로 설정됩니다.
nativeType? "button" | "submit" | "reset"
Default "button"

HTML 버튼의 네이티브 타입을 설정합니다.
