Input v1.0.0

사용자로부터 텍스트 입력을 받는 폼 컴포넌트입니다.

1

CInput 컴포넌트를 import 합니다.

JAVASCRIPT ES Module 
import { CInput } from '@linecorp/cuetify'
VUE Vue SFC 
<template>
  <CInput v-model="value" />
</template>

<script setup>
import { ref } from 'vue'
import { CInput } from '@linecorp/cuetify'

const value = ref('')
</script>
2

가장 기본적인 Input 사용 방법입니다.

VUE Basic Usage 
<CInput v-model="value" placeholder="텍스트를 입력하세요" />
3

type prop을 사용하여 다양한 입력 타입을 지정할 수 있습니다.

VUE Types 
<CInput type="text" placeholder="텍스트" />
<CInput type="email" placeholder="이메일" />
<CInput type="password" placeholder="비밀번호" />
<CInput type="number" placeholder="숫자" />
4

다양한 스타일과 기능을 조합하여 사용할 수 있습니다.

8자 이상 입력해주세요
올바른 이메일 주소를 입력하세요
VUE Advanced Usage 
<!-- 레이블과 함께 사용 -->
<CInput v-model="email" label="이메일" placeholder="이메일을 입력하세요" />

<!-- 도움말 텍스트와 함께 사용 -->
<CInput v-model="password" type="password" label="비밀번호" helperText="8자 이상 입력해주세요" />

<!-- 에러 상태 -->
<CInput v-model="email" label="이메일" :error="true" errorMessage="올바른 이메일 주소를 입력하세요" />

<!-- 비활성화 상태 -->
<CInput disabled placeholder="비활성화됨" />

Properties

Prop Type
type? "text" | "email" | "password" | "number" | "tel"
Default "text"

입력 필드의 타입을 설정합니다.
size? "small" | "medium" | "large"
Default "medium"

입력 필드의 크기를 설정합니다.
label? string
Default undefined

입력 필드 상단에 표시되는 레이블 텍스트입니다.
placeholder? string
Default undefined

입력 전 표시되는 플레이스홀더 텍스트입니다.
helperText? string
Default undefined

입력 필드 하단에 표시되는 도움말 텍스트입니다.
error? boolean
Default false

에러 상태를 표시합니다.
errorMessage? string
Default undefined

에러 상태일 때 표시되는 메시지입니다.
disabled? boolean
Default false

입력 필드를 비활성화합니다.

개요

크기

VUE Size

<CInput size="medium" placeholder="Medium" />
<CInput size="large" placeholder="Large" />

VUE Type

<CInput disabled placeholder="Disabled" />
