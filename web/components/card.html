Card v1.0.0

카드는 관련된 콘텐츠와 액션을 그룹화하는 컨테이너 컴포넌트입니다.

시작하기

1

Card 컴포넌트를 import 합니다.

JS ES Module 
import { CCard } from '@linecorp/cuetify'
VUE Vue SFC 
<template>
  <CCard>
    <p>카드 내용</p>
  </CCard>
</template>

<script setup>
import { CCard } from '@linecorp/cuetify'
</script>
2

가장 기본적인 Card 사용 방법입니다.

상품 카드

카드 컴포넌트의 기본 사용 예시입니다.

VUE Basic Usage 
<CCard>
  <p>카드 내용</p>
</CCard>
3

variant prop을 사용하여 다양한 스타일의 카드를 만들 수 있습니다.

Elevated

Outlined

Filled

VUE Variants 
<CCard variant="elevated">Elevated</CCard>
<CCard variant="outlined">Outlined</CCard>
<CCard variant="filled">Filled</CCard>
4

슬롯을 활용하여 헤더와 푸터를 추가합니다.

카드 제목

카드 내용입니다. 여기에 상세 설명이 들어갑니다.

VUE Advanced Usage 
<CCard>
  <template #header>
    <h3>카드 제목</h3>
  </template>

  <p>카드 내용입니다.</p>

  <template #footer>
    <CButton variant="outline">취소</CButton>
    <CButton variant="brandSolid">확인</CButton>
  </template>
</CCard>

Properties

Prop Type
variant? "elevated" | "outlined" | "filled"
Default "elevated"

카드의 시각적 스타일을 설정합니다.
padding? "none" | "sm" | "md" | "lg"
Default "md"

카드 내부 패딩 크기를 설정합니다.
clickable? boolean
Default false

카드를 클릭 가능한 상태로 설정합니다. hover/active 상태 스타일이 적용됩니다.
onClick? () => void

카드 클릭 시 호출되는 이벤트 핸들러입니다.
