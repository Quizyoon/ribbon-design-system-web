Grid

CSS Grid 기반의 레이아웃 시스템으로 복잡한 2차원 레이아웃을 쉽게 구성할 수 있습니다.

Basic Grid

기본 그리드 레이아웃을 생성합니다.

html Basic Grid 
<div class="gs-grid">
  <div class="gs-col">Column 1</div>
  <div class="gs-col">Column 2</div>
  <div class="gs-col">Column 3</div>
</div>

Column Count

그리드의 열 수를 지정합니다.

html Column Count 
<!-- 2 columns -->
<div class="gs-grid gs-grid-cols-2">...</div>

<!-- 3 columns -->
<div class="gs-grid gs-grid-cols-3">...</div>

<!-- 4 columns -->
<div class="gs-grid gs-grid-cols-4">...</div>

<!-- 6 columns -->
<div class="gs-grid gs-grid-cols-6">...</div>

<!-- 12 columns -->
<div class="gs-grid gs-grid-cols-12">...</div>

Gap

그리드 아이템 사이의 간격을 지정합니다.

html Grid Gap 
<!-- gap: 4px -->
<div class="gs-grid gs-gap-1">...</div>

<!-- gap: 8px -->
<div class="gs-grid gs-gap-2">...</div>

<!-- gap: 16px -->
<div class="gs-grid gs-gap-4">...</div>

<!-- gap: 24px -->
<div class="gs-grid gs-gap-6">...</div>

<!-- gap: 32px -->
<div class="gs-grid gs-gap-8">...</div>

Column Span

특정 열이 여러 열을 차지하도록 지정합니다.

html Column Span 
<div class="gs-grid gs-grid-cols-12 gs-gap-4">
  <div class="gs-col-span-8">Main Content</div>
  <div class="gs-col-span-4">Sidebar</div>
</div>
