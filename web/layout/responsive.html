Responsive

반응형 디자인을 위한 브레이크포인트와 유틸리티 클래스입니다.

Breakpoints

LINE GIFTSHOP Design System의 기본 브레이크포인트입니다.

Breakpoint Prefix Min Width CSS
Mobile sm: 640px @media (min-width: 640px)
Tablet md: 768px @media (min-width: 768px)
Desktop lg: 1024px @media (min-width: 1024px)
Large Desktop xl: 1280px @media (min-width: 1280px)

Responsive Grid

브레이크포인트별로 그리드 열 수를 변경합니다.

html Responsive Grid 
<!-- 1 column on mobile, 2 on tablet, 4 on desktop -->
<div class="gs-grid gs-grid-cols-1 md:gs-grid-cols-2 lg:gs-grid-cols-4">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
  <div>Item 4</div>
</div>

Responsive Display

브레이크포인트별로 요소를 표시하거나 숨깁니다.

html Responsive Display 
<!-- Hidden on mobile, visible on tablet and up -->
<div class="gs-hidden md:gs-block">
  Desktop Sidebar
</div>

<!-- Visible on mobile, hidden on tablet and up -->
<div class="gs-block md:gs-hidden">
  Mobile Menu Button
</div>

Responsive Spacing

브레이크포인트별로 패딩과 마진을 조정합니다.

html Responsive Spacing 
<!-- Padding: 16px on mobile, 32px on tablet, 48px on desktop -->
<div class="gs-p-4 md:gs-p-8 lg:gs-p-12">
  Content with responsive padding
</div>

Responsive Flex Direction

브레이크포인트별로 Flex 방향을 변경합니다.

html Responsive Flex 
<!-- Column on mobile, row on tablet and up -->
<div class="gs-flex gs-flex-col md:gs-flex-row gs-gap-4">
  <div>Left / Top</div>
  <div>Right / Bottom</div>
</div>
