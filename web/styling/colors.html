Colors

This guide explains how to use LINE GIFTSHOP design system color tokens in your web projects.

Installation

Install the design system package using npm or yarn.

bash 
# npm
npm install @line-giftshop/design-system

# yarn
yarn add @line-giftshop/design-system

# pnpm
pnpm add @line-giftshop/design-system

Import the CSS file into your project.

typescript 
// main.tsx or App.tsx
import '@line-giftshop/design-system/styles/colors.css';

Usage

CSS Variables

You can access color tokens using CSS variables. All tokens start with the --color- prefix.

css 
.my-component {
  /* Brand colors */
  background-color: var(--color-primary);           /* #06C755 */
  color: var(--color-text-primary);                 /* #111111 */

  /* Semantic colors */
  border-color: var(--color-border-default);        /* #E0E0E0 */

  /* Feedback colors */
  --success: var(--color-success);                   /* Green */
  --error: var(--color-error);                       /* Red */
  --warning: var(--color-warning);                   /* Orange */
}

React / JSX

You can use inline styles in React components, or import and use the token object.

tsx 
import { colors } from '@line-giftshop/design-system';

function MyComponent() {
  return (
    <div
      style={{
        // Using CSS variables
        backgroundColor: 'var(--color-primary)',

        // Using token object
        color: colors.text.primary,
        borderColor: colors.border.default,
      }}
    >
      LINE GIFTSHOP
    </div>
  );
}

Styled Components

Here is how to use tokens with Styled Components or Emotion.

tsx 
import styled from 'styled-components';
import { colors } from '@line-giftshop/design-system';

const PrimaryButton = styled.button`
  background-color: ${colors.primary};
  color: ${colors.text.inverse};
  border: none;
  padding: 12px 24px;
  border-radius: 8px;
  cursor: pointer;

  &:hover {
    background-color: ${colors.primary.hover};
  }

  &:disabled {
    background-color: ${colors.disabled.background};
    color: ${colors.disabled.text};
  }
`;

const Card = styled.div`
  background: var(--color-background-primary);
  border: 1px solid var(--color-border-default);
  border-radius: 12px;
  padding: 24px;
`;

Tailwind CSS

To use with Tailwind CSS, register custom colors in tailwind.config.js.

javascript 
// tailwind.config.js
const { tailwindColors } = require('@line-giftshop/design-system/tailwind');

module.exports = {
  theme: {
    extend: {
      colors: tailwindColors,
    },
  },
};

Now you can use tokens in Tailwind classes.

html 
<!-- Brand colors -->
<button class="bg-line-primary text-white hover:bg-line-primary-hover">
  Buy Now
</button>

<!-- Semantic colors -->
<div class="bg-line-surface border border-line-border">
  <p class="text-line-text-primary">Content</p>
</div>

<!-- Status colors -->
<span class="text-line-success">Payment Complete</span>
<span class="text-line-error">Error Occurred</span>

Using in LIFF Environment

In the LIFF (LINE Front-end Framework) environment, colors are applied considering the characteristics of the browser within the LINE app.

tsx 
import { useLiff } from '@line-giftshop/design-system/hooks';
import { colors } from '@line-giftshop/design-system';

function App() {
  const { isInClient, colorScheme } = useLiff();

  // Check if running inside LINE app
  const statusBarColor = isInClient
    ? colors.primary
    : colors.background.primary;

  // Match LINE app color scheme
  useEffect(() => {
    if (colorScheme === 'dark') {
      document.documentElement.setAttribute('data-theme', 'dark');
    }
  }, [colorScheme]);

  return (
    <div style={{ backgroundColor: statusBarColor }}>
      {/* ... */}
    </div>
  );
}

LIFF Status Bar Color Setting

You can set the status bar color to the brand color within the LINE app.

tsx 
import liff from '@line/liff';

// Set status bar to LINE GIFTSHOP brand color
liff.init({
  liffId: 'YOUR_LIFF_ID'
}).then(() => {
  // Set status bar color on iOS
  const meta = document.createElement('meta');
  meta.name = 'theme-color';
  meta.content = '#06C755'; // LINE Green
  document.head.appendChild(meta);
});

Dark Mode Implementation

The design system supports both light and dark modes. Switch themes using the data-theme attribute.

tsx 
import { useState, useEffect } from 'react';

function useTheme() {
  const [theme, setTheme] = useState<'light' | 'dark'>('light');

  useEffect(() => {
    // Detect system settings
    const mediaQuery = window.matchMedia('(prefers-color-scheme: dark)');
    setTheme(mediaQuery.matches ? 'dark' : 'light');

    const handler = (e: MediaQueryListEvent) => {
      setTheme(e.matches ? 'dark' : 'light');
    };

    mediaQuery.addEventListener('change', handler);
    return () => mediaQuery.removeEventListener('change', handler);
  }, []);

  useEffect(() => {
    document.documentElement.setAttribute('data-theme', theme);
  }, [theme]);

  const toggleTheme = () => {
    setTheme(prev => prev === 'light' ? 'dark' : 'light');
  };

  return { theme, toggleTheme };
}

Automatic Dark Mode Color Switching

When using CSS variables, colors are automatically changed when switching themes.

css 
/* Component styles - automatically changes based on theme */
.card {
  background: var(--color-background-primary);   /* Light: #FFFFFF, Dark: #1E1E1E */
  color: var(--color-text-primary);              /* Light: #111111, Dark: #FFFFFF */
  border: 1px solid var(--color-border-default); /* auto switch */
}

/* Brand colors remain the same regardless of theme */
.brand-text {
  color: var(--color-primary);  /* Always #06C755 */
}

TypeScript Type Definitions

Color token types are provided for type safety.

typescript 
import type {
  ColorToken,
  SemanticColor,
  PaletteColor
} from '@line-giftshop/design-system';

// Color token type
type ColorToken =
  | 'primary'
  | 'secondary'
  | 'success'
  | 'warning'
  | 'error'
  | 'info';

// Semantic color type
interface SemanticColor {
  text: {
    primary: string;
    secondary: string;
    disabled: string;
    inverse: string;
  };
  background: {
    primary: string;
    secondary: string;
    elevated: string;
  };
  border: {
    default: string;
    strong: string;
  };
}

// Usage example
function getColorValue(token: ColorToken): string {
  return colors[token];
}
