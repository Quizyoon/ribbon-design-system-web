Typography

How to use Typography CSS variables and classes.

CSS Implementation

/* Title XL */ .text-title-xl { font-size: var(--typography-font-size-t9); line-height: var(--typography-line-height-title-t9-en); font-weight: var(--typography-font-weight-bold); } /* Body MD */ .text-body-md { font-size: var(--typography-font-size-t4); line-height: var(--typography-line-height-body-t4); font-weight: var(--typography-font-weight-regular); } /* Caption SM */ .text-caption-sm { font-size: var(--typography-font-size-t1); line-height: var(--typography-line-height-body-t1-en); font-weight: var(--typography-font-weight-regular); }

Usage

<!-- HTML Usage --> <h1 class="text-title-2xl">Page Title</h1> <h2 class="text-title-xl">Section Title</h2> <h3 class="text-title-lg">Card Title</h3> <p class="text-body-md">Body text</p> <span class="text-caption-sm">Label</span>

Multi-language Support

/* English */ .text-title-xl-en { font-size: var(--typography-font-size-t9); line-height: var(--typography-line-height-title-t9-en); font-weight: var(--typography-font-weight-bold); font-family: var(--typography-font-family-en); } /* Traditional Chinese */ .text-title-xl-tw { font-size: var(--typography-font-size-t9); line-height: var(--typography-line-height-title-t9-tw); font-weight: var(--typography-font-weight-bold); font-family: var(--typography-font-family-tw); } /* Thai */ .text-title-xl-th { font-size: var(--typography-font-size-t9); line-height: var(--typography-line-height-title-t9-th); font-weight: var(--typography-font-weight-bold); font-family: var(--typography-font-family-th); }
