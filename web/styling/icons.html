Icons

Learn how to use icons in your LINE GIFTSHOP web application.

Icon Component

Basic Usage

Import and use the Icon component with the icon name.

Icon Usage 
import { Icon } from '@line-giftshop/design-system'

// Basic usage
<Icon name="home" />

// With size
<Icon name="search" size="lg" />

// With color
<Icon name="heart" color="primary" />

Icon Sizes

Available Sizes

Icons support multiple sizes based on the design system's dimension tokens.

2xs
xs
sm
md
lg
xl
Icon Sizes 
<Icon name="home" size="2xs" />  {/* 14px */}
<Icon name="home" size="xs" />   {/* 16px */}
<Icon name="home" size="sm" />   {/* 20px */}
<Icon name="home" size="md" />   {/* 22px */}
<Icon name="home" size="lg" />   {/* 24px */}
<Icon name="home" size="xl" />   {/* 32px */}

Icon Colors

Color Variants

Apply semantic colors to icons using the color prop.

Icon Colors 
<Icon name="heart" />                    {/* Default */}
<Icon name="heart" color="primary" />    {/* Green */}
<Icon name="heart" color="error" />      {/* Red */}
<Icon name="heart" color="info" />       {/* Blue */}
