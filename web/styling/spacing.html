Spacing

This guide explains how to use LINE GIFTSHOP design system spacing tokens in your web projects.

Usage Principles CSS Variables Best Practices Common Patterns Tailwind Integration

Usage Principles

Recommended

/* Using Component Token */ .button { padding: var(--component-button-padding-m); } /* Using Semantic Token */ .card { padding: var(--spacing-m); }

Not Recommended

/* Using Dimension directly */ .element { padding: var(--dimension-x3); }

CSS Variables

Spacing Tokens

Spacing CSS variables to use in your project.

:root { /* Spacing Tokens */ --spacing-2xs: 2px; --spacing-xs: 4px; --spacing-s: 8px; --spacing-m: 12px; --spacing-m-plus: 14px; --spacing-l: 16px; --spacing-xl: 20px; }

Component Tokens

Component-specific spacing CSS variables.

:root { /* Button Gap */ --component-button-gap-2: var(--spacing-2xs); --component-button-gap-4: var(--spacing-xs); /* Button Padding */ --component-button-padding-xs: var(--spacing-s); --component-button-padding-s: var(--spacing-m); --component-button-padding-m: var(--spacing-m-plus); --component-button-padding-l: var(--spacing-l); --component-button-padding-xl: var(--spacing-xl); }

Best Practices

1. Token Priority

Token Priority
  1. Component Token - For component development
  2. Semantic Token - For general layouts
  3. Dimension - Do not use

2. Consistency

Use the same token for the same purpose.

/* Use the same token for the same purpose */ .card-header, .card-body, .card-footer { padding: var(--spacing-m); }

3. Responsive Design

Utilize tokens for responsive design.

.container { padding: var(--spacing-m); } @media (min-width: 768px) { .container { padding: var(--spacing-l); } }

Common Patterns

Button Styling

.button { padding: var(--component-button-padding-m); gap: var(--component-button-gap-4); } .button--small { padding: var(--component-button-padding-s); gap: var(--component-button-gap-2); }

Card Layout

.card { padding: var(--spacing-m); margin-bottom: var(--spacing-l); gap: var(--spacing-s); }

Form Elements

.form-field { margin-bottom: var(--spacing-m); } .form-label { margin-bottom: var(--spacing-xs); }

Tailwind Integration

Configuration

Add spacing tokens to tailwind.config.js.

// tailwind.config.js module.exports = { theme: { extend: { spacing: { '2xs': 'var(--spacing-2xs)', 'xs': 'var(--spacing-xs)', 's': 'var(--spacing-s)', 'm': 'var(--spacing-m)', 'm-plus': 'var(--spacing-m-plus)', 'l': 'var(--spacing-l)', 'xl': 'var(--spacing-xl)', } } } }

Usage

<!-- Using Tailwind classes --> <div class="p-m mb-l"> Component </div>
