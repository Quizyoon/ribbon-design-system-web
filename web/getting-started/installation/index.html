Installation

LINE Gift Shop 디자인 시스템을 설치하는 방법입니다.

1

LINE Gift Shop 디자인 시스템 패키지를 설치합니다.

Terminal 
npm install @line-giftshop/design-system
yarn add @line-giftshop/design-system
pnpm add @line-giftshop/design-system
bun add @line-giftshop/design-system
2

프레임워크별 스타일을 import합니다.

Next.js

typescript app/layout.tsx 
import '@line-giftshop/design-system/styles.css';

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="ko">
      <body>{children}</body>
    </html>
  );
}

Vue 3

javascript main.js 
import { createApp } from 'vue';
import App from './App.vue';
import '@line-giftshop/design-system/styles.css';

const app = createApp(App);
app.mount('#app');

Vite

javascript main.js 
import '@line-giftshop/design-system/styles.css';
3

페이지에서 컴포넌트를 사용합니다.

typescript app/page.tsx 
import { Button, Card } from '@line-giftshop/design-system';

export default function Home() {
  return (
    <main>
      <Card>
        <h1>Welcome to LINE Gift Shop</h1>
        <Button variant="primary">Get Started</Button>
      </Card>
    </main>
  );
}
