CDN

Use LINE GIFTSHOP Design System via CDN without any build tools.

1

HTML 문서에 CSS 파일을 직접 추가하세요.

html CSS Link 
<link rel="stylesheet" href="https://cdn.line-giftshop.com/design-system/latest/styles.css">
2

CDN을 통해 디자인 시스템을 사용하는 전체 HTML 예시입니다.

html index.html 
<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>LINE GIFTSHOP</title>

  <!-- LINE GIFTSHOP Design System CSS -->
  <link rel="stylesheet" href="https://cdn.line-giftshop.com/design-system/latest/styles.css">
</head>
<body>
  <div class="gs-container">
    <!-- Card Component -->
    <div class="gs-card">
      <div class="gs-card__header">
        <h2 class="gs-card__title">LINE GIFTSHOP에 오신 것을 환영합니다</h2>
      </div>
      <div class="gs-card__body">
        <p>빌드 도구 없이 CDN으로 디자인 시스템을 사용하세요.</p>

        <!-- Button Components -->
        <button class="gs-button gs-button--primary">
          시작하기
        </button>
        <button class="gs-button gs-button--secondary">
          더 알아보기
        </button>
      </div>
    </div>

    <!-- Input Component -->
    <div class="gs-input-group">
      <label class="gs-label">이메일</label>
      <input type="email" class="gs-input" placeholder="이메일을 입력하세요">
    </div>
  </div>
</body>
</html>
