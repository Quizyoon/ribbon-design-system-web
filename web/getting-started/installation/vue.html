Vue 3

Install and configure LINE GIFTSHOP Design System in your Vue 3 project.

1

Vue 3 호환 디자인 시스템 패키지를 설치하세요.

terminal Terminal 
npm install @line-giftshop/design-system-vue
2

메인 진입 파일에서 디자인 시스템을 Vue 플러그인으로 등록하세요.

typescript src/main.ts 
import { createApp } from 'vue'
import App from './App.vue'
import { GiftshopPlugin } from '@line-giftshop/design-system-vue'
import '@line-giftshop/design-system-vue/styles.css'

const app = createApp(App)

app.use(GiftshopPlugin)

app.mount('#app')
3

Vue 컴포넌트에서 컴포넌트 사용을 시작하세요.

버튼 컴포넌트

vue src/components/ButtonExample.vue 
<template>
  <div>
    <LGSButton variant="primary" @click="handleClick">
      Primary Button
    </LGSButton>
    <LGSButton variant="secondary">
      Secondary Button
    </LGSButton>
  </div>
</template>

<script setup lang="ts">
const handleClick = () => {
  console.log('Button clicked!')
}
</script>

입력 컴포넌트

vue src/components/InputExample.vue 
<template>
  <div>
    <LGSInput
      v-model="name"
      label="이름"
      placeholder="이름을 입력하세요"
    />
    <LGSInput
      v-model="email"
      type="email"
      label="이메일"
      placeholder="이메일을 입력하세요"
    />
  </div>
</template>

<script setup lang="ts">
import { ref } from 'vue'

const name = ref('')
const email = ref('')
</script>
