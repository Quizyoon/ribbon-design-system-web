Next.js

Install and configure LINE GIFTSHOP Design System in your Next.js project.

1

LINE GIFTSHOP 디자인 시스템 패키지를 설치하세요.

terminal Terminal 
npm install @line-giftshop/design-system
2

루트 레이아웃 파일에서 스타일을 import하세요.

App Router

typescript app/layout.tsx 
import "@line-giftshop/design-system/styles.css";

export default function RootLayout({
  children,
}: {
  children: React.ReactNode
}) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  )
}

Pages Router

typescript pages/_app.tsx 
import "@line-giftshop/design-system/styles.css";
import type { AppProps } from 'next/app'

export default function App({ Component, pageProps }: AppProps) {
  return <Component {...pageProps} />
}
3

페이지에서 컴포넌트 사용을 시작하세요.

typescript app/page.tsx 
import { Button, Card } from '@line-giftshop/design-system'

export default function Home() {
  return (
    <main>
      <Card>
        <h1>Welcome to LINE GIFTSHOP</h1>
        <Button variant="primary">
          Get Started
        </Button>
      </Card>
    </main>
  )
}
