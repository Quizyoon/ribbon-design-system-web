Vite

Install and configure LINE GIFTSHOP Design System in your Vite project.

1

Vite 프로젝트에 디자인 시스템 패키지를 설치하세요.

terminal Terminal 
npm install @line-giftshop/design-system
2

메인 진입 파일에서 CSS를 import하세요.

typescript src/main.tsx 
import React from 'react'
import ReactDOM from 'react-dom/client'
import App from './App'
import '@line-giftshop/design-system/styles.css'

ReactDOM.createRoot(document.getElementById('root')!).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
)
3

Vite 애플리케이션에서 컴포넌트 사용을 시작하세요.

typescript src/App.tsx 
import { Button, Card } from '@line-giftshop/design-system'

function App() {
  return (
    <main>
      <Card>
        <h1>Welcome to LINE GIFTSHOP</h1>
        <Button variant="primary">
          Get Started
        </Button>
      </Card>
    </main>
  )
}

export default App
