Quick Start

Build your first LINE GIFTSHOP web application in minutes.

1

Install the LINE GIFTSHOP design system package.

Terminal 
npm install @line-giftshop/design-system
yarn add @line-giftshop/design-system
pnpm add @line-giftshop/design-system
bun add @line-giftshop/design-system
2

Import the design system styles and components.

typescript app/layout.tsx 
import "@line-giftshop/design-system/styles.css";
3

Start using components in your project.

typescript app/page.tsx 
import { Button } from '@line-giftshop/design-system'

export default function Home() {
  return <Button variant="primary">Click me</Button>
}

Usage Methods

There are multiple ways to use the design system.

Components

Import and use React components directly.

typescript Components.tsx 
import { Button, Card, TextField } from '@line-giftshop/design-system'

CSS Variables

Use design tokens as CSS custom properties.

css styles.css 
.custom-element {
  color: var(--gs-color-primary);
  padding: var(--gs-spacing-4);
  border-radius: var(--gs-radius-md);
}

Utility Classes

Apply pre-built utility classes for common styles.

html index.html 
<div class="gs-flex gs-gap-4 gs-p-6">
  <span class="gs-text-primary gs-font-bold">Hello</span>
</div>

Here are some common usage examples.

Gift Card

Send a special gift to your loved ones with LINE GIFTSHOP.

typescript CardExample.tsx 
<Card>
  <Card.Header>
    <Card.Title>Gift Card</Card.Title>
  </Card.Header>
  <Card.Body>
    <p>Send a special gift to your loved ones.</p>
  </Card.Body>
  <Card.Footer>
    <Button variant="primary" size="sm">Send Gift</Button>
  </Card.Footer>
</Card>
