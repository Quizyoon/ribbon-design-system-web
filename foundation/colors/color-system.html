Color System

The color system provides a structured approach to using colors consistently across the LINE GIFTSHOP platform. It defines how colors are organized, named, and applied throughout the design system.

Color System Color Roles Color Palette Primitive Colors

Color Architecture

Systematic Color Structure

Our color system is built on a foundation of carefully selected hues that work harmoniously together. Each color serves a specific purpose and follows consistent naming conventions.

The system includes primitive colors, semantic tokens, and component-specific color applications to ensure flexibility and maintainability.

Colors are organized into three tiers: Core primitives, Semantic tokens, and Component tokens for maximum flexibility.
