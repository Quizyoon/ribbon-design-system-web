Icon

The icon system provides a consistent set of visual symbols for the LINE GIFTSHOP platform. Icons help users quickly understand actions, navigation, and content types.

Principles Sizes Usage

Icon System

Design Principles

Icons are designed with consistent stroke weights, corner radius, and optical balance. They maintain clarity at all sizes and work seamlessly across light and dark themes.

Icon Sizes

Icons are available in multiple sizes to accommodate different use cases. Standard sizes include 16px, 20px, 24px, and 32px, based on the 4px grid system.

Usage Guidelines

Use icons consistently throughout the interface. Pair icons with labels when possible to improve accessibility. Ensure sufficient contrast between icons and backgrounds.
