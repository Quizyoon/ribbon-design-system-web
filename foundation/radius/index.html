Foundation > Radius

Radius

Border radius tokens define the roundness of corners for UI elements. Based on dimension values, they ensure consistent corner styling across all components.

Scale Usage

Radius System

Consistent Corner Styling

Radius tokens range from xs (2px) to full (999px for pill shapes). Each value is based on the dimension token system, ensuring mathematical consistency. Use smaller radii for subtle rounding and larger values for prominent rounded elements.

Radius Tokens

Border radius values from xs to full for consistent corner styling.

Token Value

$radius.s 4px
