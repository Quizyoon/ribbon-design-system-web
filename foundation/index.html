Foundation

Foundation is the core building blocks of the design system. It provides consistent design tokens for colors, typography, spacing, and more to ensure visual consistency across the LINE GIFTSHOP platform.

Token Structure Token Hierarchy Usage

Design Token System

Token Structure

Foundation consists of multiple categories that work together to create a cohesive design language. Colors define the visual palette, Typography handles text styling across languages, Dimension provides base measurement units, and Spacing ensures consistent layout rhythm.

Token Hierarchy

Foundation tokens follow a three-tier hierarchy: Core (Primitive) tokens contain raw values like colors and sizes, Semantic tokens define purpose-based values like backgrounds and text colors, and Component tokens provide specific values for UI components. This hierarchy ensures maintainability and consistency.

Usage Guidelines

Always use Semantic or Component tokens in your implementations instead of Core tokens directly. This approach ensures that theme changes and updates propagate correctly throughout the system. All measurements follow a 4px grid system for consistent alignment and spacing.
