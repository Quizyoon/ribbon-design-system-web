Typography

The typography system provides consistent text styles across multiple languages in the LINE GIFTSHOP platform. It defines how text is styled, sized, and weighted throughout the design system.

Title Body Core

Multi-language Typography

Language-optimized Text Styles

Typography tokens are designed to support multiple languages including English, Korean, Chinese, and Thai. Each language has optimized font families and line heights to ensure readability and visual consistency across all locales.

Title Styles

Title styles are used for headings and emphasized text. They support sizes from T2 to T10, with both Regular and Bold weights. Title styles have tighter line heights optimized for short, impactful text.

Body Styles

Body styles are designed for paragraph text and longer content. They support sizes from T1 to T5, with line heights optimized for comfortable reading. Body text uses more generous line spacing to improve readability in longer passages.

Core Tokens

Core typography tokens include Font Family, Font Weight, Font Size, and Line Height. These primitive values are combined to create the Style tokens. Core tokens should not be used directly - use Style tokens instead for consistent typography.
