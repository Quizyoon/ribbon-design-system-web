Foundation > Dimension

Dimension

The dimension system provides primitive values that serve as the foundation for all sizing in the LINE GIFTSHOP platform. Based on a 4px grid system, it ensures visual consistency across all components.

4px Grid Scale Usage

4px Grid Foundation

Consistent Spacing System

Dimension is the foundation of all size values in the design system. It provides a consistent size system based on a 4px grid, serving as base values for Spacing, Border Radius, Icon Size, and more. As a Core Token, it is referenced by Semantic and Component Tokens to maintain visual harmony.

4px Grid System

All Dimension values are defined based on 4px. This consistent base unit ensures that all spacing and sizing across the platform maintains visual rhythm and alignment.

Scale Values

Dimension provides a comprehensive scale from x0_5 (2px) to x16 (64px) and beyond. Each scale value is calculated by multiplying the base unit (4px) by the scale factor.

Usage Principle

Dimension is a Core Token and should not be used directly in components. Instead, use Semantic Tokens (like Spacing) or Component Tokens that reference Dimension values. This abstraction layer allows for easier maintenance and theme switching.
